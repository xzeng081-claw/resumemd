## 📌 产品概述

**木及简历** - 免费 Markdown 在线简历，专业简历制作工具

**核心定位：** 以最简单的方式来写好简历，只需专注内容本身而无需关注排版

**数据：** 累计制作 100000+ 份简历

---

## 🎯 核心功能

### 1. 多主题选择 ⭐
**描述：** 一份内容，多份简历，多种样式和排版尽收眼底

**价值：**
- 无需重复编辑内容
- 快速切换不同风格
- 适配不同公司/岗位

### 2. 智能一页 ⭐⭐⭐
**描述：** 告别简历一页写不下，两页却太长的尴尬境地

**价值：**
- 自动调整内容到一页 A4
- 智能调整：字体大小、行距、页边距
- 解决常见简历排版痛点

**技术实现：**
- 内容长度检测算法
- 动态缩放逻辑
- 优先级：内容 > 格式

### 3. 智能检测 ⭐⭐
**描述：** 错别字、常见专业英文词汇、标点误用一网打尽

**价值：**
- 提升简历专业性
- 避免低级错误
- 技术术语规范

**检测范围：**
- 中文错别字
- 专业英文词汇（技术术语）
- 标点符号误用

### 4. 插件模式
**描述：** 可插拔式的设置，随时给你一片清爽的编辑页面

**价值：**
- 按需启用功能
- 减少界面干扰
- 个性化定制

### 5. 多种格式导出
**支持格式：**
- PDF
- Markdown
- PNG

## 🎯 可借鉴功能

### P0（核心差异化）

#### 1. 智能一页 ⭐⭐⭐
**理由：**
- 独特卖点，技术有挑战
- 解决真实痛点（简历页数尴尬）
- 竞品少有实现

**实现方案：**
- 内容长度检测
- 动态调整：字体、行距、页边距
- 用户可手动覆盖

**Issue：** #10

---

### P1（增值功能）

#### 2. 智能检测 ⭐⭐
**理由：**
- AI 辅助内容优化
- 提升简历质量
- 技术术语词库可积累

**实现方案：**
- 中文拼写检查：pycorrector
- 技术术语词库：自定义
- 可选：接入 AI API

**Issue：** #11

#### 3. 多种格式导出 ⭐
**理由：**
- 用户多场景需求
- 技术实现简单

**支持格式：**
- PDF（已有 #5）
- Markdown（直接下载）
- PNG（html2canvas）
- DOCX（Pandoc，可选）

**Issue：** #12

---

### P2（可选）

#### 4. 插件模式
**理由：** 高级定制需求，非核心

#### 5. 简历内推/面试神器
**理由：** 商业化方向，暂不聚焦

## 🚀 我们的差异化

### 1. 云端存储 + 自有部署
- 木及简历：SaaS 云端
- ResumeMD：自有服务器部署
- 优势：数据可控、隐私保护

### 2. ATS 优化
- 木及简历：未强调
- ResumeMD：中文 ATS 友好格式
- 优势：通过 HR 系统筛选

### 3. AI 辅助扩展
- 木及简历：基础检测
- ResumeMD：可扩展 AI 内容优化
- 优势：智匠团队 AI 能力

### 4. 开发者友好
- 木及简历：通用型
- ResumeMD：代码高亮、技术术语优化
- 优势：更懂开发者
