## 📌 产品概述

### 产品名称
**ResumeMD** - Markdown 在线简历生成器

### 产品定位
面向开发者的轻量级在线简历工具，支持 Markdown 编辑和 PDF 导出。

### 目标用户
- 曾星（个人使用）
- 未来可扩展至开发者群体

### 核心价值
| 价值点 | 描述 |
|--------|------|
| 开发者友好 | Markdown 编辑，无需学习成本 |
| 快速生成 | 一键导出 PDF，方便投递 |
| 独立展示 | 云服务器部署，专业域名 |
| 简洁高效 | 无多余功能，专注简历本身 |

## 📋 功能需求

### F1：Markdown 编辑器（P0）
**描述：** 在线编辑简历内容

**功能点：**
- [ ] 支持标准 Markdown 语法
- [ ] 左右分栏：左侧编辑，右侧预览
- [ ] 支持代码高亮
- [ ] 支持表格、列表、链接

**验收标准：**
- 编辑器加载时间 < 1s
- 输入延迟 < 100ms
- 支持常见 Markdown 语法

---

### F2：实时预览（P0）
**描述：** 编辑时实时渲染预览

**功能点：**
- [ ] 编辑区输入后即时更新预览
- [ ] 支持响应式布局（桌面/移动端）

**验收标准：**
- 预览更新延迟 < 200ms
- 移动端显示正常

---

### F3：PDF 导出（P0）
**描述：** 服务端生成 PDF 文件

**功能点：**
- [ ] 点击按钮触发 PDF 生成
- [ ] 自动下载 PDF 文件
- [ ] PDF 格式规整，适合 A4 打印

**验收标准：**
- 生成时间 < 3s
- PDF 文件大小 < 1MB
- 打印无裁切问题

---

### F4：在线展示页（P0）
**描述：** 独立 URL 展示简历

**功能点：**
- [ ] 固定 URL（如 `/resume`）
- [ ] 简洁专业的 UI
- [ ] 支持分享

**验收标准：**
- 页面加载时间 < 2s
- Lighthouse 分数 > 90

---

### F5：数据持久化（P1）
**描述：** 云端存储简历内容

**功能点：**
- [ ] 自动保存到服务器
- [ ] 支持手动保存
- [ ] 历史记录/版本回溯

**验收标准：**
- 保存成功有提示
- 刷新页面不丢失数据

---

### F6：模板切换（P1）
**描述：** 多套简历模板可选

**功能点：**
- [ ] 至少 2 套模板
- [ ] 一键切换，内容不变

**验收标准：**
- 切换后格式正常
- PDF 导出适配模板
