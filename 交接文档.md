### 2. 竞品分析（PM 完成）

**深度分析 3 个项目：**
1. **oh-my-cv** - 功能最丰富，本地优先
2. **markdown-resume** - ATS 优化，DOCX 导出
3. **resume** - 一键部署，组件化

**核心结论：**
- 市场空白：云端存储 + ATS 优化 + 中文支持
- 差异化：自有服务器部署、多简历管理、AI 辅助

**技术选型建议：**
| 层级 | 推荐方案 | 状态 |
|------|---------|------|
| 前端 | Next.js 14 + TypeScript | ✅ 已采用 |
| 后端 | FastAPI + SQLAlchemy | ✅ 已采用 |
| 数据库 | PostgreSQL | ✅ 已采用 |
| PDF 生成 | Puppeteer | ✅ 已采用 |
| 缓存 | Redis | ✅ 已采用 |
| 部署 | Docker Compose | ✅ 已采用 |

## 📋 关键决策点

### 1. 存储方案
**决策：** PostgreSQL（已实现）  
**理由：** 结构化数据、支持查询、易备份

### 2. PDF 生成方案
**决策：** Puppeteer 服务端渲染  
**理由：** 格式稳定、可控性强

### 3. 编辑器选型
**待确认：** Monaco Editor vs TipTap  
- **Monaco Editor：** VS Code 同款，开发者友好
- **TipTap：** 富文本 + Markdown，更灵活

**建议：** 先用 Monaco Editor（更符合开发者习惯）

### 4. 部署方案
**决策：** Docker Compose 一键部署  
**理由：** 环境隔离、易迁移、易维护
